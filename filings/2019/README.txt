Nike SEC Filings 2019 - Sample Files for Testing
==============================================

This directory contains sample SEC filing documents for Nike Inc. for the year 2019.
This data is created for testing the SEC filings extraction and analysis pipeline.

FILE TYPES:
-----------
• PDF files: SEC forms (10-K, 10-Q, 8-K, DEF 14A, etc.)
• XLS/XLSX files: Financial data supplements

IMPORTANT DOCUMENTS:
-------------------
• Form 10-K: Annual report with comprehensive business overview
• Form 10-Q: Quarterly reports with financial statements
• Form 8-K: Current reports for material events
• DEF 14A: Proxy statements for shareholder meetings
• Financial Supplements: Detailed quarterly financial data

For questions about these filings, visit:
https://investors.nike.com/
